# รายงานผลการวิเคราะห์ชุดข้อมูล Titanic

เอกสารนี้สรุปขั้นตอนการวิเคราะห์ข้อมูลและผลลัพธ์จากการสร้างโมเดลเพื่อทำนายการรอดชีวิตของผู้โดยสาร Titanic

## 1. ข้อมูลทั่วไป (Dataset Overview)
- **แหล่งข้อมูล**: Titanic-Dataset.csv
- **เป้าหมาย**: ทำนายการรอดชีวิต (Survived) : 0 = ไม่รอด, 1 = รอด
- **คำอธิบายคอลัมน์ (Data Dictionary)**:
  - **PassengerId**: รหัสผู้โดยสาร
  - **Survived**: การรอดชีวิต (0 = ไม่รอด, 1 = รอด)
  - **Pclass**: ชั้นที่นั่ง (1 = ชั้น 1, 2 = ชั้น 2, 3 = ชั้น 3)
  - **Name**: ชื่อผู้โดยสาร
  - **Sex**: เพศ
  - **Age**: อายุ
  - **SibSp**: จำนวนพี่น้อง/คู่สมรสที่เดินทางมาด้วย
  - **Parch**: จำนวนบิดามารดา/ลูกที่เดินทางมาด้วย
  - **Ticket**: หมายเลขตั๋ว
  - **Fare**: ค่าโดยสาร
  - **Cabin**: หมายเลขห้องพัก
  - **Embarked**: ท่าเรือที่ขึ้น (C = Cherbourg, Q = Queenstown, S = Southampton)

## 2. การเตรียมข้อมูล (Preprocessing)
จากการสำรวจข้อมูลพบว่ามีค่าที่หายไป (Missing Values) ในบางคอลัมน์ จึงได้ทำการจัดการดังนี้:
- **Age**: แทนค่าที่หายไปด้วยค่ามัธยฐาน (Median) ของผู้โดยสารทั้งหมด
- **Embarked**: แทนค่าที่หายไปด้วยค่าฐานนิยม (Mode) หรือค่าที่พบได้บ่อยที่สุด (ซึ่งคือ 'S')
- **Cabin**: ตัดคอลัมน์นี้ออกเนื่องจากมีข้อมูลหายไปเป็นจำนวนมาก
- **Feature Selection**: เลือกใช้เฉพาะคอลัมน์ที่มีความสำคัญและตัด Name, Ticket, PassengerId ออก
- **Encoding**: แปลงข้อมูลเพศ (Sex) และท่าเรือ (Embarked) ให้เป็นตัวเลขเพื่อให้โมเดลสามารถนำไปคำนวณได้

## 3. การสร้างโมเดลและการประเมินผล (Modeling & Evaluation)
เราได้ทำการแบ่งข้อมูลเป็นชุดสอน (Training Set) 80% และชุดทดสอบ (Test Set) 20% และทดสอบกับ 2 โมเดล ได้ผลลัพธ์ดังนี้:

### 3.1 Decision Tree Classifier
- **ความถูกต้อง (Accuracy)**: 79.89%
- **ข้อสังเกต**: โมเดลนี้มีความเรียบง่ายและสามารถอธิบายได้ง่าย แต่อาจเกิดปัญหา Overfitting ได้หากไม่จำกัดความลึกของต้นไม้

### 3.2 Random Forest Classifier
- **ความถูกต้อง (Accuracy)**: 82.68%
- **ข้อสังเกต**: ให้ผลลัพธ์ที่แม่นยำกว่า Decision Tree เนื่องจากเป็นการรวมผลจากต้นไม้หลายต้น (Ensemble Learning) ช่วยลดความแปรปรวน (Variance) และป้องกัน Overfitting ได้ดีกว่า

## 4. สรุปผล (Conclusion)
จากการทดสอบพบว่า **Random Forest Classifier** ให้ประสิทธิภาพที่ดีกว่า Decision Tree ในการทำนายการรอดชีวิตของผู้โดยสาร Titanic บนชุดข้อมูลชุดนี้ โดยมีค่าความแม่นยำอยู่ที่ประมาณ 82.68%

*หมายเหตุ: ผลลัพธ์นี้อ้างอิงจากการรันครั้งล่าสุด ค่าอาจมีการเปลี่ยนแปลงเล็กน้อยขึ้นอยู่กับการสุ่มแบ่งข้อมูล (Random State)*
